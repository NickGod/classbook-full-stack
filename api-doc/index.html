classbook API

The classbook API.

Default response content-types: application/json
Schemes: https

Summary

Path Operation Description
/user/auth POST

Sign up for a new user.
/user/auth/sign_in POST

Log in as a returning user.
/user/auth/sign_out DELETE

Log out from current session.

Paths

Sign up for a new user.

POST /user/auth

This endpoint sends a request for registration to the server.

Uses default content-types: application/json

201 Created

The new user has been created successfully.

User
422

The server failed to create the new user because the form has ill data.

Error
Log in as a returning user.

POST /user/auth/sign_in

This endpoint sends a login request to the server.

Uses default content-types: application/json

200 OK

The new user has been created successfully.

User
401 Unauthorized

The login fails because the password is not correct.

Error
422

The server fails to create the new user because the form has ill data.

Error
Log out from current session.

DELETE /user/auth/sign_out

This endpoints sends a logout request to the server.

Uses default content-types: application/json

200 OK

The user has signed out from the current session.
